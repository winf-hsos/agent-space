Quickstart

  1. Create a Telegram bot via @BotFather
  2. Paste the token into the Token Value field
  3. Message the bot once, then get your chat ID from getUpdates
  4. Set a workdir path and optionally add an AGENTS.md there
  5. Save, then start the bridge from the sidebar
